# Halfo
Halfo is a python script designed to analyze any data you throw at it.
This includes:

  * files
  * directories
  * numbers
  * URLs
  * python scripts
  * words
  * concepts
 
 ## Dependencies
 To work properly halfo needs:
 
  * linux distro
  * python3
  * nmap
  * traceroute
  * file (command)
